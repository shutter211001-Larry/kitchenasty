# 🔒 Privacy Policy

**Effective date:** February 2025\
**Last updated:** February 2025

This privacy policy explains how the KitchenAsty documentation website (hosted on GitHub Pages) handles visitor data.

## 1️⃣ Who We Are

KitchenAsty is an open-source restaurant ordering platform. This privacy policy applies to the **documentation website** at [mighty840.github.io/kitchenasty](https://mighty840.github.io/kitchenasty/), not to individual KitchenAsty installations operated by restaurants.

- 📦 **Project**: KitchenAsty (open-source)
- 🔗 **Repository**: [github.com/mighty840/kitchenasty](https://github.com/mighty840/kitchenasty)

## 2️⃣ What Data We Collect

### 🚫 Data We Do NOT Collect

This documentation website:

- 🍪 Does **not** use cookies
- 📊 Does **not** use analytics or tracking scripts (no Google Analytics, no Plausible, no Matomo)
- 👤 Does **not** collect personal information
- 🔑 Does **not** require user accounts or login
- 📢 Does **not** use advertising or third-party ad networks
- 💬 Does **not** use comment systems or social media plugins

### 🌐 Data Collected by GitHub Pages

This website is hosted on [GitHub Pages](https://pages.github.com/). GitHub may collect technical data as part of serving the website. According to [GitHub's Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement):

- 🖥️ **IP addresses**: GitHub may log IP addresses of visitors for security purposes
- 📋 **Request data**: Standard HTTP request information (browser type, referring page, date/time of visit) may be logged

We do not have access to this data. It is governed by GitHub's own privacy policies.

### 🔍 Local Search

The documentation search feature runs entirely in your browser. Search queries are **not** sent to any server.

Since we do not collect any data, there is no data to retain. For data collected by GitHub Pages, refer to GitHub's data retention policies.

## 5️⃣ Your Rights

Under the GDPR and similar privacy regulations, you have the right to:

- 🔍 **Access**: Request information about any personal data processed about you
- ✏️ **Rectification**: Request correction of inaccurate data
- 🗑️ **Erasure**: Request deletion of your personal data
- ⏸️ **Restriction**: Request restriction of processing
- 📤 **Data portability**: Request a copy of your data in a structured format
- 🚫 **Objection**: Object to data processing

Since this website does not collect personal data, these rights are primarily relevant to data collected by GitHub Pages. To exercise your rights regarding data collected by GitHub, please contact GitHub directly through their [privacy contact form](https://support.github.com/contact/privacy).

## 6️⃣ Children's Privacy

This documentation website does not knowingly collect any personal information from children under the age of 16. The website is a technical documentation resource and is not directed at children.

We may update this privacy policy from time to time. Changes will be reflected in the "Last updated" date at the top of this page and committed to the [project repository](https://github.com/mighty840/kitchenasty).
